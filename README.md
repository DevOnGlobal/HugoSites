Hugo sites For Employees

*Prerequisites:*
- Hugo
- Git
- Access to Github
- Access to Azure Portal

## Install Hugo on Microsoft:
*Prequisites:*
- Powershell
- scoop

## Steps
- Run Powershell
- install scoop 
`Invoke-Expression (New-Object System.Net.WebClient).DownloadString('https://get.scoop.sh')`
- install hugo 
`scoop install hugo-extended`
